# Logos — Maya Protege

Arquivos vetoriais (SVG) da marca **Maya Protege**. Todos são escaláveis sem perda de qualidade.

## Variantes

| Arquivo | Layout | Cor | Quando usar |
|---------|--------|-----|-------------|
| `maya-protege-symbol.svg` | Símbolo isolado | Verde `#14D366` | Favicon, app icon, avatar, selos |
| `maya-protege-horizontal-dark.svg` | Horizontal (símbolo + texto) | Texto escuro `#0C332A` | Sobre fundos **claros** |
| `maya-protege-horizontal-light.svg` | Horizontal (símbolo + texto) | Texto claro `#EFFEF5` | Sobre fundos **escuros** |
| `maya-protege-stacked-dark.svg` | Empilhado (símbolo + texto em 2 linhas) | Texto escuro `#0C332A` | Espaços mais quadrados, fundos **claros** |
| `maya-protege-stacked-light.svg` | Empilhado (símbolo + texto em 2 linhas) | Texto claro `#EFFEF5` | Espaços mais quadrados, fundos **escuros** |

> **Sufixo `-dark` / `-light`** refere-se à cor da **logo**, não do fundo.
> Use `-dark` (logo escura) em fundos claros e `-light` (logo clara) em fundos escuros.

## Uso

### HTML
```html
<img src="assets/logos/maya-protege-horizontal-dark.svg" alt="Maya Protege" height="40">
```

### Favicon
```html
<link rel="icon" type="image/svg+xml" href="assets/logos/maya-protege-symbol.svg">
```

## Cores da marca

- Símbolo (versão vibrante): `#14D366` — `--color-sea-green-500`
- Símbolo (versão horizontal/empilhada): `#43C970`
- Texto escuro: `#0C332A` — `--color-timber-green-950`
- Texto claro: `#EFFEF5`

## Diretrizes

- Preserve uma área de respiro ao redor da logo equivalente à altura do símbolo.
- Não distorça, rotacione ou recolora a logo fora das variantes oficiais.
- Para fundos de imagem, prefira a variante `-light` com contraste suficiente.
